Airbnb - Bangkok Analysis

This project involves cleaning Airbnb data from Bangkok using R, focusing on analyzing room types, availability for short vs. long-term rentals, hosts with multiple listings, and an interactive neighborhood plot. The dataset contains 23,651 observations and 18 variables.
